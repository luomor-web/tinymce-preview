TinyMCE Preview Plugin
v1.0.0
开箱即用 · 支持 Vue 2/3

多端实时预览
编辑所见即所得

为 TinyMCE 打造的多端预览插件，一键切换手机、平板、电脑视图，支持横屏旋转，完美兼容 Vue 2 / Vue 3。

手机预览
375×812，还原真实手机视口，验证移动端排版效果
平板预览
768×1024，iPad 级别视口，检查中屏幕布局
PC 预览
1280×800，桌面级完整展示，支持宽屏内容预览
横竖屏切换
一键旋转设备，验证横屏下的内容适配情况
在线演示
💡 在下方编写内容后，点击工具栏中的 👁 预览 按钮，或使用下方的快捷按钮多端预览
使用方式
// 1. 引入插件脚本
<script src="/plugins/multipreview/plugin.js"></script>

// 2. 初始化 TinyMCE
tinymce.init({
  selector: '#editor',
  plugins: ['multipreview', '...'],
  toolbar: 'bold italic | multipreview',
  // 可选：自定义预览样式
  multipreview: {
    customStyles: 'body { background: #f9fafb; }'
  }
})
<!-- Vue 2 Options API -->
<template>
  <TinymcePreviewEditor
    v-model="content"
    :height="500"
    plugin-path="/plugins/multipreview/plugin.js"
    :show-external-trigger="true"
    ref="editor"
  />
</template>

<script>
import TinymcePreviewEditor from './TinymcePreviewEditor.vue'

export default {
  components: { TinymcePreviewEditor },
  data() {
    return { content: '' }
  },
  methods: {
    preview() {
      this.$refs.editor.openPreview()
    }
  }
}
</script>
<!-- Vue 3 Composition API -->
<template>
  <TinymcePreviewEditor
    v-model="content"
    :height="500"
    ref="editorRef"
  />
  <button @click="editorRef.openPreview()">预览</button>
</template>

<script setup>
import { ref } from 'vue'
import TinymcePreviewEditor from './TinymcePreviewEditor.vue3.vue'

const editorRef = ref(null)
const content = ref('')
</script>
